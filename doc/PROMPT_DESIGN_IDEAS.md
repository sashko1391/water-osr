# Промпт для генерації ідей дизайну сайту води «Остреченська»

---

## Контекст

Ти — senior UI/UX дизайнер та creative director з досвідом у luxury/premium брендах напоїв. Тобі потрібно запропонувати концепцію сучасного, стильного, іммерсивного вебсайту для українського бренду мінеральної води.

## Про бренд

- **Назва:** Остреченська
- **Слоган:** «Остреченська — визнана якість поліського краю»
- **Компанія:** ТОВ «Нептун», Чернігівська область, Україна
- **Продукт:** мінеральна лікувально-столова вода з глибини 830 метрів (гідрокарбонатно-хлоридно-натрієва), солодкі газовані напої (9 смаків), питна негазована вода «Сіверська»
- **Регіон:** екологічно чистий Поліський край
- **Сертифікація:** ISO 9001-2000
- **Особливість:** вода розливається безпосередньо на місці видобутку з джерела

## Цільова аудиторія

- B2B: оптові покупці, магазини, HoReCa
- B2C: кінцеві споживачі, які цінують натуральність та якість
- Вік: 25–55 років
- Географія: Україна, переважно Чернігівська область та сусідні регіони

## Вимоги до сайту

### Обов'язкові секції
1. **Hero** — повноекранний перший екран з WOW-ефектом
2. **Про воду** — джерело, глибина 830 м, мінеральний склад, процес видобутку
3. **Каталог продукції** — 3 категорії: мінеральна газована, солодка газована (9 смаків), питна негазована
4. **Про компанію** — історія, виробництво, сертифікація ISO
5. **Доставка** — 3 склади (Мена, Чернігів, Шостка), Нова Пошта, власний транспорт
6. **Контакти** — телефони, email, адреси, карта з 3 точками

### Технічний стек (vanilla, без фреймворків)
- HTML5, CSS3, vanilla JavaScript
- Бібліотеки через CDN: GSAP + ScrollTrigger, Three.js, Lenis (smooth scroll)
- Web Audio API для звукових ефектів
- WebGL шейдери для води

## Завдання

Запропонуй **детальну креативну концепцію** сайту, яка включає:

### 1. Візуальна концепція
- Кольорова палітра (hex-коди) — прив'язка до води, природи, Полісся
- Типографіка — які шрифти (Google Fonts), розміри, ієрархія
- Стиль UI — glassmorphism, neomorphism, мінімалізм, або мікс?
- Фотостиль — який mood, фільтри, композиція

### 2. Анімації та ефекти (деталізуй кожен)
- Hero секція — як зустрічаємо користувача?
- Scroll-triggered анімації — що відбувається при скролі?
- Hover-ефекти — карточки продуктів, кнопки, навігація
- Мікроанімації — лоадер, курсор, переходи
- Частинки — бульбашки, краплі, блиски?
- Вода — шейдери, рипли, хвилі, переливи?
- Звук — ambient водяний фон, інтерактивні звуки?
- Parallax — які шари, яка глибина?
- 3D-елементи — пляшка, краплі, хвилі?

### 3. UX/Flow
- Як користувач проходить через сайт?
- Як працює навігація?
- Як виглядає каталог — сітка, каруселька, 3D?
- Mobile-first чи desktop-first?
- Як адаптуються ефекти під мобільні?

### 4. Секція-кілер
Запропонуй одну унікальну секцію або фічу, яка зробить цей сайт вірусним і запам'ятовуваним. Щось, чого немає у конкурентів. Поясни детально як це реалізувати технічно.

### 5. Референси
Наведи 5-7 реальних сайтів-референсів із подібною естетикою (luxury water, premium beverages, immersive experiences), поясни що саме взяти від кожного.

## Формат відповіді

Структурована відповідь з розділами. Кожен ефект/елемент описуй конкретно: що, де, як, на якому тригері. Уникай загальних фраз типу «красивий дизайн» — давай конкретику: CSS properties, анімаційні криві, тривалість, easing.

## Обмеження

- Сайт має працювати без фреймворків (React/Vue/etc.) — чистий HTML/CSS/JS + CDN бібліотеки
- Performance: <3s load time, 60fps анімації
- Accessibility: контраст тексту, keyboard navigation, reduced-motion media query
- SEO: семантичний HTML, meta tags, Open Graph
- Не e-commerce: це презентаційний сайт, без кошика/оплати (є контактна форма для замовлень)
